# 🚀 Nova AI Assistant

An AI-powered Voice Assistant built with **React**, **FastAPI**, **Python**, **Gemini AI**, **Pandas**, and **Matplotlib**. Nova can interact using voice commands, analyze CSV datasets, generate charts and PDF reports, provide AI-powered responses, fetch weather information, and deliver an interactive dashboard experience.

---

## ✨ Features

### 🤖 AI Assistant

* Voice Recognition (Speech-to-Text)
* Text-to-Speech Responses
* AI-powered Conversation using Gemini
* Wikipedia Search
* Weather Information
* Open Websites (Google, YouTube, GitHub)
* Play Songs on YouTube
* Current Time

### 📊 Data Analysis

* Load CSV files using Pandas
* Automatic Data Analysis
* Average, Total, Highest & Lowest Calculations
* Statistical Summary (`describe()`)
* Dynamic Insights & Observations

### 📈 Data Visualization

* Bar Chart
* Scatter Plot
* Correlation Heatmap
* Charts automatically saved

### 📄 Report Generation

* Professional PDF Report
* Analysis Summary
* Charts Included

### 🌐 Web Dashboard

* React Frontend
* FastAPI Backend
* REST API Integration
* Responsive UI
* Modern Dashboard Layout

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yash4200/Nova-AI.git
```

```bash
cd Nova-AI
```

---

## Backend

```bash
cd backend
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate

Windows

```bash
venv\Scripts\activate
```

Install Requirements

```bash
pip install -r requirements.txt
```

Run FastAPI

```bash
uvicorn app:app --reload
```

Run Voice Assistant

```bash
python nova.py
```

---

## Frontend

```bash
cd frontend
```

```bash
npm install
```

```bash
npm run dev
```
